Functional Contact Form

This contact form uses Python, the email module, the smtplib module, and Flask to not only send an email message to me but also to send a confirmation email to you, additionally, it writes your information and message down into a csv file located on the server.

First the server runs a function that turns the form data into a dictionary. From there, it writes that data to a csv by calling a function that utilizes the csv module and pythons built in file handling.
